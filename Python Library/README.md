# 📚 Python Libraries Cheat Sheet

A curated list of powerful and popular Python libraries organized by category.

---

## 🔢 Data Science & Analysis

- **NumPy** – Powerful numerical operations and array handling
- **pandas** – Data manipulation and analysis with DataFrames
- **Matplotlib** / **Seaborn** – Data visualization
- **SciPy** – Scientific computing (optimization, stats, etc.)
- **Statsmodels** – Statistical modeling and hypothesis testing

---

## 🤖 Machine Learning & AI

- **scikit-learn** – Core ML algorithms (classification, regression, clustering)
- **TensorFlow** / **PyTorch** – Deep learning frameworks
- **XGBoost** / **LightGBM** – High-performance gradient boosting
- **Transformers (by Hugging Face)** – NLP and pre-trained models

---

## 🌐 Web Development

- **Flask** – Lightweight web framework
- **Django** – Full-featured web framework with ORM
- **FastAPI** – Fast web APIs with automatic docs (based on Pydantic)
- **Requests** – Easy HTTP requests

---

## 🛠️ Automation & Scripting

- **os** / **shutil** / **subprocess** – Built-in system interaction
- **pyautogui** – GUI automation (mouse/keyboard control)
- **schedule** – Job scheduling (like cron for Python)
- **selenium** – Web browser automation

---

## 🧪 Testing & Debugging

- **unittest** / **pytest** – Testing frameworks
- **pdb** – Built-in debugger
- **logging** – Logging and tracking events

---

## 🔒 Cybersecurity / Web Scraping

- **BeautifulSoup** – HTML/XML parsing
- **Scrapy** – Web scraping framework
- **paramiko** – SSH and SFTP in Python
- **hashlib** – Built-in cryptographic hashing

---

## 📦 Packaging & Environment Management

- **pip** – Package installer
- **virtualenv** / **venv** – Virtual environments
- **setuptools** – Packaging Python projects
- **poetry** – Dependency management and packaging
